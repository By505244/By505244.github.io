Day0

早上 6:40 起床。lcr 展现超强码力秒切了去年 noip 的 A 题。

考场上：

A 题第一眼：卧槽，A 题考平衡树。

第二眼：完了，怎么考 kmp。昨晚做梦许愿不要考最后还是考到了。

第三眼：排个序就完了，是什么傻逼题目。

B 题想了一会儿的并查集，拓扑排序解法，没想出来。考场上认为处理中间状态的情况比较棘手。果断暴力后跳题。

C 题一开始想了一个假的贪心。又不断修正了几个小时。最后发现假的贪心可以过特殊性质，有 $55$ 分！欢天喜地打了暴力和贪心。但样例没过，时间不太够，先跳 D 题打暴力。

D 题，为给 C 题留时间，一眼看出了 $\mathcal{O}(4^n)$ 解法。然后写完了发现读错题了。被迫修正。打过了样例。

C 题，发现读入成了没有特殊性质的文件了。换成有特殊性质的文件之后就过了。

考前最后 1s：发现 C 数组只开了 $10^5$，惊险。

在火车站在周围人的提醒下发现 A 题会存在 $n=1$ 的情况，而且出题人明确指出在数据中含有 $n=1$ 的点。没特判。寄。抱着侥幸去云斗交了几发。A 爆了 $10$ 分。D 暴力不知为何挂了。$0$ 分。在此特地感谢小方在动车上提供了靠近电源插座的位置，wjx 热心提供了流量。

$\text{score}=90+20+55+0=165$
